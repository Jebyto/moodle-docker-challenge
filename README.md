# Moodle Docker Challenge

Ambiente Moodle containerizado para gerenciamento acadêmico básico.

## Objetivo

Configurar um ambiente local com Moodle, MySQL, carga inicial de alunos/disciplinas e uma customização simples via plugin Moodle.

## Stack prevista

- Moodle
- MySQL
- Docker
- Docker Compose
- PHP
- Plugin local Moodle

## Como executar

Instruções serão detalhadas conforme a implementação evoluir.

## Decisões técnicas

- Uso de Docker Compose para orquestrar Moodle e MySQL.
- Uso de variáveis de ambiente via `.env`.
- Customização prevista via plugin local.
